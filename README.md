# StackDock

Local Windows-first workspace manager for terminals, git, and quick edits.

## Dev

```bash
npm install
npm run dev
```

## Unified VS Code theme support

Settings → Looks & feel has one Theme dropdown for the whole app. StackDock can import VS Code theme JSON/JSONC files and maps VS Code `colors` into dashboard, workspace, panel, terminal, and editor UI variables. `tokenColors` are also mapped into Monaco syntax highlighting.

Full VS Code extension installation, icon themes, and exact TextMate grammar fidelity are not included.
